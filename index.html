🎯 프로젝트 개요

한국아트크래프트협회의 공식 홈페이지를 구축하여, 협회 활동과 교육 프로그램을 온라인으로 안내하고 회원 및 잠재 수강생에게 전문적인 이미지를 전달합니다.

💡 직접 관리 가능 — Notion에서 콘텐츠를 수정하면 홈페이지에 자동 반영됩니다. 별도 개발 요청 없이 협회에서 직접 관리할 수 있습니다.

인프라 관리 (유지보수 포함)

  • 호스팅 운영 — Vercel 서버에서 홈페이지가 항상 정상 작동하도록 관리
  • 도메인 관리 — 협회 희망 도메인 신규 등록·연결·갱신을 대행 협회 미보유 → 신규 취득 대행
  • SSL 보안 인증서 — HTTPS 보안 연결을 자동 유지
  • 보안 업데이트 — 웹 프레임워크 보안 패치를 정기 적용
  • 자동 배포 — Notion에서 내용을 수정하면 홈페이지에 자동 반영

기술 지원

  • 버그·오류 수정 — 문제가 생기면 신속하게 해결
  • 콘텐츠 수정 도움 — Notion 사용이 어려우면 대신 수정
  • 사용법 안내 — 카카오톡·전화로 Notion CMS 사용법 안내
✨ TIP — 처음 1개월은 무상 유지보수 기간입니다. 이 기간 동안 사용법을 충분히 익히신 후 유지보수 계약 여부를 결정하실 수 있습니다.
